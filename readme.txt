Name: Like&Subscribe
(Developed by Python 3.7)

주의사항
- chrome창을 종료하고 다시 실행하고 싶을 때는, 녹화하는 창에 커서를 움직여 
  +모양이 되었을 때 키보드에서 q를 눌러 먼저 종료시키고, 
  후에 '종료' 버튼을 눌러주세요. 
  (녹화하는 창이 멈추는 것이 정상입니다)
- 유튜브 영상에 들어가야 각종 기능을 실행할 수 있습니다. 
- 'Hand Control을 위한 설명서'도 읽어야 직접 조작할 수 있습니다. 
- 필요한 라이브러리는 mediapipe, opencv, undetected-chromedriver와 pyautogui가 되겠습니다. 
(python파일에서 실행할 때만 필요합니다.)

기본적인 실행 방법
1. 구글 로그인용 아이디 비번 입력
2. 실행 누르기
3. 크롬 창에서 자동 로그인하고 youtube.com에 접속
4. manual에 있는 손모양을 인식시키면서 이용하면 됨
